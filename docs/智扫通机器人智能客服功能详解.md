## 1. 项目概述

**智扫通机器人智能客服**是一个基于AI大模型的RAG（检索增强生成）与智能体开发项目，专门针对扫地机器人领域。项目集成了：

- **RAG检索增强生成**：从本地知识库检索相关信息并生成准确回答
- **多模态OCR**：支持PDF、Word、图片等多种格式文档的自动OCR识别
- **ReactAgent智能体**：具备自主思考、工具调用能力的AI客服
- **MCP远程工具调用**：支持通过Model Context Protocol调用远程服务
- **Web交互界面**：基于Streamlit的友好用户界面

**技术栈**：
- Python 3.8+
- LangChain（Agent框架）
- Chroma（向量数据库）
- 通义千问/Qwen（大语言模型）
- Streamlit（Web界面）
- FastMCP（MCP协议实现）

## 3. RAG检索增强生成系统

### 3.1 RAG工作流程

RAG（Retrieval-Augmented Generation）即检索增强生成，工作流程如下：

```
用户提问 → 向量检索 → 获取相关文档 → 组装提示词 → AI生成回答
```

#### 3.1.1 核心模块：[rag_service.py](rag/rag_service.py)

```python
class RagSummarizeService:
    def __init__(self):
        # 1. 初始化向量数据库检索器
        self.vector_store = VectorStoreService()
        self.retriever = self.vector_store.get_retriever()
        
        # 2. 加载RAG专用提示词模板
        self.prompt_text = load_rag_prompts()
        self.prompt_template = PromptTemplate.from_template(self.prompt_text)
        
        # 3. 构建处理链：提示词 → 模型 → 输出解析
        self.model = chat_model
        self.chain = self.prompt_template | self.model | StrOutputParser()
```

#### 3.1.2 RAG执行流程

```python
def rag_summarize(self, query: str) -> str:
    # 第一步：检索相关文档
    context_docs = self.retriever_docs(query)
    
    # 第二步：拼接参考资料文本
    context_parts = []
    for i, doc in enumerate(context_docs, start=1):
        context_parts.append(
            f"【参考资料{i}】: 参考资料：{doc.page_content} | 参考元数据：{doc.metadata}"
        )
    context = "\n".join(context_parts)
    
    # 第三步：提交给AI生成回答
    return self.chain.invoke({"input": query, "context": context})
```

#### 3.1.3 RAG提示词模板

RAG提示词位于 `prompts/rag_prompts.txt`，核心结构：
```
你是一个扫地机器人专家助手。我将给你一些参考资料，你需要根据这些资料回答问题。

参考资料：
{context}

用户问题：
{input}

要求：
1. 基于参考资料回答，不要编造信息
2. 如果参考资料不足以回答，明确说明
3. 回答简洁专业
```

```python
def _check_md5_exists(self, md5_hex: str) -> bool:
    """检查文件是否已入库（避免重复处理）"""
    md5_file = get_abs_path(chroma_conf["md5_hex_store"])
    
    if not os.path.exists(md5_file):
        return False
    
    with open(md5_file, "r", encoding="utf-8") as f:
        for line in f:
            if line.strip() == md5_hex:
                return True
    return False

def _save_md5(self, md5_hex: str):
    """入库成功后记录MD5"""
    with open(md5_file, "a", encoding="utf-8") as f:
        f.write(md5_hex + "\n")
```

```python
def recognize_image(self, image_path: str) -> Optional[str]:
    try:
        image_url = self._image_to_base64(image_path)  # 转为base64
        message = HumanMessage(content=[
            {"type": "text", "text": OCR_PROMPT},
            {"type": "image_url", "image_url": {"url": image_url}},
        ])
        text = self.vision_model.invoke([message]).content.strip()
```

```python
def recognize_pdf_pages(self, pdf_path: str, pages: Optional[List[int]] = None):
    """扫描版PDF：每页转图片再OCR"""
    try:
        from pdf2image import convert_from_path
        images = convert_from_path(pdf_path)
        
        results = []
        with tempfile.TemporaryDirectory() as tmp:
            for i, image in enumerate(images):
                page_num = i + 1
                img_path = os.path.join(tmp, f"page_{page_num}.jpg")
                image.save(img_path, "JPEG")
                text = self.recognize_image(img_path)
                if text:
                    results.append((page_num, text))
        return results
    except Exception as e:
        logger.error(f"[OCR] PDF {pdf_path} 失败: {e}")
        return []
```

```python
def load_docx(filepath: str) -> list[Document]:
    documents = []
    
    try:
        doc = DocxDocument(filepath)
        
        # 1. 提取文字
        text_parts = [para.text for para in doc.paragraphs if para.text.strip()]
        if text_parts:
            full_text = "\n\n".join(text_parts)
            documents.append(Document(
                page_content=full_text,
                metadata={"source": filepath, "type": "docx_text"}
            ))
        
        # 2. 提取并OCR图片
        image_docs = _extract_images_from_docx(filepath)
        if image_docs:
            documents.extend(image_docs)
        
        return documents
    except Exception as e:
        logger.error(f"[Word] {filepath} 失败: {e}", exc_info=True)
        return []
```

```python
def _extract_images_from_docx(docx_path: str) -> list[Document]:
    """从Word文档中提取图片并OCR识别"""
    try:
        with zipfile.ZipFile(docx_path, 'r') as zip_ref:
            # 查找word/media/下的图片
            image_files = [
                f for f in zip_ref.namelist()
                if f.startswith('word/media/') and
                f.lower().endswith(('.png', '.jpg', '.jpeg', '.gif', '.bmp'))
            ]
            
            for idx, img_path_in_zip in enumerate(image_files, start=1):
                # 读取图片数据
                img_data = zip_ref.read(img_path_in_zip)
                
                # 保存为临时文件
                with tempfile.NamedTemporaryFile(suffix='.jpg', delete=False) as tmp:
                    tmp.write(img_data)
                    tmp_path = tmp.name
                
                try:
                    # 使用OCR服务识别图片
                    text = ocr_service.recognize_image(tmp_path)
                    if text and len(text.strip()) > 10:
                        documents.append(Document(
                            page_content=f"【Word文档内图片{idx}】\n{text}",
                            metadata={
                                "source": docx_path,
                                "image_name": os.path.basename(img_path_in_zip),
                                "type": "docx_image_ocr"
                            }
                        ))
                finally:
                    if os.path.exists(tmp_path):
                        os.remove(tmp_path)
        
        return documents
    except Exception as e:
        logger.error(f"[Word图片] {docx_path} 失败: {e}", exc_info=True)
        return []
```

```python
def _load_single_mcp(self, server_url: str) -> List:
    """连接单个MCP服务并获取其工具"""
    from zs_mcp.mcp_client import MCPClient
    
    async def _run():
        client = MCPClient(server_url)
        await client.connect()
        tools = await client.get_tools()
        self._mcp_clients.append(client)
        return tools
    
    try:
        return asyncio.run(_run())  # 运行异步代码
    except Exception as e:
        logger.debug(f"[Agent] 单服 MCP 连接失败: {e}")
        return []

def _load_multi_mcp(self, configs: Dict[str, Dict[str, Any]]) -> List:
    """连接多个MCP服务并获取所有工具"""
    from zs_mcp.multi_server_client import MultiServerMCPClient
    
    async def _run():
        client = MultiServerMCPClient(configs)
        await client.connect_all()
        tools = await client.get_tools()
        self._mcp_clients.append(client)
        return tools
    
    try:
        return asyncio.run(_run())
    except Exception as e:
        logger.debug(f"[Agent] 多服 MCP 连接失败: {e}")
        return []
```

#### 5.2.3 消息处理与流式响应

```python
def _build_messages(self, history: List[Dict], current_query: str) -> List:
    """把历史记录和当前问题转换成Agent能理解的消息格式"""
    messages = []
    for msg in history:
        if msg["role"] == "user":
            messages.append(HumanMessage(content=msg["content"]))
        elif msg["role"] == "assistant":
            messages.append(AIMessage(content=msg["content"]))
    messages.append(HumanMessage(content=current_query))
    return messages

def _stream_response(self, messages: List):
    """提交消息给Agent，以流式方式返回回答（逐块输出）"""
    input_dict = {"messages": messages}
    for chunk in self.agent.stream(
        input_dict, stream_mode="values", context={"report": False}
    ):
        latest = chunk["messages"][-1]
        if latest.content:
            yield latest.content.strip() + "\n"
```

```python
def monitor_tool(

    tool_name = request.tool_call['name']    # 获取工具名

        request.runtime.context["report"] = True
    
    return result
```

```python
        results = list(ddgs.text(query, max_results=5))
        if not results:

        summaries = []
        for r in results[:5]:
            title = r.get("title", "")
            body = r.get("body", "")
            href = r.get("href", "")
            summaries.append(f"标题：{title}\n摘要：{body}\n链接：{href}")
        return "\n---\n".join(summaries)
    except Exception as e:
        return f"搜索失败: {e}"
```

```python
class MCPClient:
    def __init__(self, server_url: str = "http://localhost:8001/sse"):
        self.server_url = server_url
        self._client: Optional[Client] = None
        self.tools: List[Tool] = []
    
    async def connect(self) -> Optional[Client]:
        if not FASTMCP_AVAILABLE:
            logger.error("[MCP] fastmcp 未安装，无法连接 MCP 服务器")
            return None
        
        if self._client is None:
            logger.info(f"[MCP] 正在连接 {self.server_url}")
            self._client = Client(transport=self.server_url)
            await self._client.__aenter__()  # 建立连接
            logger.info("[MCP] 连接成功")
        return self._client
    
    async def get_tools(self) -> List[Tool]:
        if not FASTMCP_AVAILABLE:
            return []
        
        client = await self.connect()
        if client is None:
            return []
        
        tools_info = await client.list_tools()  # 向服务端请求工具列表
        logger.info(f"[MCP] 获取到 {len(tools_info)} 个远程工具")
        
        # 把 MCP 工具格式转换为 LangChain Tool 格式
        self.tools = mcp_tools_to_langchain(client, tools_info)
        return self.tools
```

#### 6.2.3 工具格式转换器

```python
# zs_mcp/tool_adapter.py
def mcp_tools_to_langchain(client: Client, tools_info: List[Any], name_prefix="", desc_prefix="") -> List[Tool]:
    langchain_tools = []
    
    for tool_info in tools_info:
        tool_name = tool_info.name
        exposed_name = f"{name_prefix}{tool_name}"  # 加前缀避免命名冲突
        base_desc = tool_info.description or f"远程工具: {tool_name}"
        description = f"{desc_prefix}{base_desc}"
        
        # 定义异步函数，调用远程MCP服务
        async def call_remote(*, _client=client, _name=tool_name, **kwargs):
            try:
                return await _client.call_tool(_name, kwargs)  # 实际调用远程服务
            except Exception as e:
                return f"调用 {_name} 失败: {e}"
        
        # 创建LangChain Tool对象
        langchain_tools.append(
            Tool(
                name=exposed_name,
                description=description,
                func=None,           # 同步函数（不用）
                coroutine=call_remote,  # 异步函数（Agent调用时用）
            )
        )
    
    return langchain_tools
```

```python
def load_documents(self):
    # 1. 确定要处理哪些文件类型
    supported_exts = tuple(get_supported_extensions())
    
    # 2. 列出data/下所有符合条件的文件
    data_dir = get_abs_path(chroma_conf["data_path"])
    file_list = list_files_by_extension(data_dir, supported_exts)
    
    if not file_list:
        logger.warning(f"[知识库] {data_dir} 下未找到支持的文件")
        return
    
    logger.info(f"[知识库] 发现 {len(file_list)} 个文件待处理")
    
    # 3. 逐个处理文件
    for filepath in file_list:
        # 计算文件MD5，已处理过的跳过
        md5_hex = get_file_md5_hex(filepath)
        if not md5_hex:
            logger.warning(f"[知识库] 跳过: {filepath} (MD5计算失败)")
            continue
        
        if self._check_md5_exists(md5_hex):
            logger.info(f"[知识库] 跳过: {filepath} (已处理)")
            continue
        
        try:
            # 加载文档内容
            documents = load_document(filepath)
            if not documents:
                logger.warning(f"[知识库] 跳过: {filepath} (无有效内容)")
                continue
            
            # 把长文档切成小段
            split_docs = self.splitter.split_documents(documents)
            if not split_docs:
                logger.warning(f"[知识库] 跳过: {filepath} (分片后无内容)")
                continue
            
            # 存入向量数据库（自动转成向量）
            self.vector_store.add_documents(split_docs)
            
            # 记录已处理的MD5
            self._save_md5(md5_hex)
            logger.info(f"[知识库] 入库成功: {filepath} ({len(split_docs)}个片段)")
            
        except Exception as e:
            logger.error(f"[知识库] 入库失败: {filepath} 错误: {e}", exc_info=True)
            continue
```

```python
def get_file_md5_hex(filepath: str) -> Optional[str]:
    """计算文件的MD5哈希值（用于去重）"""
    try:
        with open(filepath, "rb") as f:
            file_hash = hashlib.md5()
            chunk = f.read(8192)
            while chunk:
                file_hash.update(chunk)
                chunk = f.read(8192)
        return file_hash.hexdigest()
    except Exception as e:
        logger.error(f"[文件工具] 计算MD5失败 {filepath}: {e}")
        return None

def list_files_by_extension(directory: str, extensions: tuple) -> list[str]:
    """列出目录下所有指定扩展名的文件"""
    file_list = []
    for root, _, files in os.walk(directory):
        for file in files:
            if file.lower().endswith(extensions):
                file_list.append(os.path.join(root, file))
    return file_list
```

## 10. 完整工作流程

### 10.1 知识库构建流程

```
1. 用户将文档放入data/目录
   ↓
2. 运行向量数据库服务：python rag/vector_store.py
   ↓
3. 系统扫描data/目录，计算每个文件的MD5
   ↓
4. 检查MD5记录文件，跳过已处理的文件
   ↓
5. 对未处理的文件：
   ├── PDF文件：提取文本层 + OCR识别图片
   ├── Word文件：提取段落 + OCR识别图片
   ├── 图片文件：直接OCR识别
   └── TXT文件：直接加载
   ↓
6. 将内容分片（每片200字，重叠20字）
   ↓
7. 将分片转换为向量，存入Chroma数据库
   ↓
8. 记录文件MD5到md5.text
```

### 10.2 用户查询处理流程

```
1. 用户在Web界面输入问题
   ↓
2. ReactAgent接收问题，分析是否需要工具调用
   ↓
3. 如果需要RAG检索：
   ├── 向量数据库检索相关文档
   ├── 组装RAG提示词（问题+参考资料）
   ├── 调用AI模型生成回答
   └── 返回给用户
   ↓
4. 如果需要联网搜索：
   ├── 调用web_search工具
   ├── 访问DuckDuckGo获取实时信息
   ├── 格式化搜索结果
   └── Agent根据结果生成回答
   ↓
5. 如果需要其他工具（天气、报告等）：
   ├── 调用对应工具函数
   ├── 获取工具返回结果
   └── Agent整合结果生成回答
   ↓
6. 流式返回回答给Web界面
```

```
1. 用户选择"MCP模式"

   ↓

   ↓

   ↓
```
